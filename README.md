## 👨‍💻 About Study

본 스터디는 단순 문제 풀이를 넘어  
**"왜 이 풀이를 선택했는가"를 설명할 수 있는 실력**을 목표로 합니다.

## 🎯 Goals

- 알고리즘 문제 해결 능력 향상
- 구현 능력 및 코드 품질 개선
- 발표 및 설명 능력 강화
- 코드 리뷰 기반 협업 경험

## 🧭 Study Process

### 1️⃣ Weekly Tasks
- 백준 / 리트 코드 / 프로그래머스 4~5문제 풀이
- AtCoder ABC 참여

### 2️⃣ 발표
- 시간 / 공간 복잡도
- 사용 알고리즘
- 선택 이유
- 핵심 아이디어

### 3️⃣ Code Management
- GitHub PR 기반 코드 제출
- Notion 알고리즘 정리

## 🛠️ Rules

- 최소 30분 이상 고민하기
- 코드 주석 꼼꼼히 작성하기
- 풀이 과정을 설명할 수 있도록 준비하기

## 📌 Commit Convention

```bash
[플랫폼] 이름(이니셜) - 문제 이름 or 번호
ex)
[BOJ] JJ - 요세푸스
[LC] JJ - Rotting Oranges
[PG] JJ - 타겟 넘버
[ABC453] JJ - A
[SS] JJ - 돌아가는 팔각 의자
```

## 🧭 Study Process

### 1️⃣ Weekly Tasks
- 백준 / 프로그래머스 4~5문제 풀이
- AtCoder ABC 참여

### 2️⃣ 발표
발표 시 포함 내용:
- 시간 / 공간 복잡도
- 사용 알고리즘
- 선택 이유
- 핵심 아이디어

📌 ABC 기준
- 푼 문제 + 다음 문제까지 준비
- A/B 문제는 간단 브리핑

### 3️⃣ Code Management
- GitHub PR 기반 코드 제출
- Notion 알고리즘 정리
